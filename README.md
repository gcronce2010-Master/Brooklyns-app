# Brooklyn's Craft Studio 🎨

A vibrant, playful digital art studio designed for creative exploration. Express yourself with professional-grade drawing tools and overcome artist's block with AI-powered inspiration.

## ✨ Features

- **Digital Drawing Canvas**: A smooth, responsive canvas supporting mouse and touch interactions via Pointer Events.
- **Advanced Brush Toolkit**:
  - Multiple brush types: Round, Square, and Marker.
  - Adjustable brush size and opacity.
  - Dedicated eraser tool.
- **Vibrant Color Palette**: Choose from a curated selection of colors or mix your own using the custom color picker.
- **AI Inspiration Engine**: Powered by Google Gemini via Genkit, generate unique art concepts, themes, and prompts based on your mood.
- **Studio Gallery**: Save your masterpieces to your local studio gallery, allowing you to load or manage your previous works.
- **Export & Share**: Download your artwork as high-quality PNGs or share them directly from your browser.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Integration**: [Genkit](https://firebase.google.com/docs/genkit) with Google Gemini
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Set up environment variables**:
   Create a `.env` file and add your `GOOGLE_GENAI_API_KEY`.

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Explore the Studio**:
   Open [http://localhost:9002](http://localhost:9002) in your browser to start creating!

## 🎨 Design Philosophy

Brooklyn's Craft Studio uses a playful, high-contrast HSL-based color scheme with soft rounded corners and deep shadows to create a tactile, "craft-like" digital experience.
